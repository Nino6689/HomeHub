Frequently Asked Questions

How does Guardian elderly monitoring work?

Guardian uses discreet smart sensors to monitor daily routines without cameras invading privacy. Motion sensors detect activity patterns, door sensors track movements, and the system alerts family members via smartphone if unusual patterns occur - like no movement by a certain time or a door left open overnight.

Is Guardian suitable for parents with dementia?

Yes, Guardian is particularly helpful for early-stage dementia care. Features include medication reminders, wandering alerts if doors open at unusual times, activity monitoring, and optional GPS tracking for peace of mind. The system requires no input from the elderly person - it works passively in the background.

How much does elderly monitoring cost in Sussex?

Guardian packages start from around £500 for basic activity monitoring (motion sensors, door sensors, smartphone app). More comprehensive packages with fall detection, medication reminders, and additional sensors range from £800-1,500. There are no monthly fees - you own the equipment outright.

Do you install Guardian systems across all of Sussex?

Yes, we install Guardian elderly monitoring systems throughout Sussex including Horsham, Crawley, Brighton, Worthing, Eastbourne, Haywards Heath, Burgess Hill, Chichester, Bognor Regis, Lewes, and all surrounding villages. We're based in Horsham with easy access across the county.
